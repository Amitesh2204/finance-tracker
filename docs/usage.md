# Savings Tracker Documentation

## Overview
This app helps you track multiple savings types and expenses. All data is stored in your browser using IndexedDB.

## How to Use
- Add a new entry by selecting the type, entering the amount, and picking a date.
- View savings and expenses graphs in the Overview section.
- All entries are listed in the History section.

## Data Storage
No data leaves your device. Everything is stored locally for privacy and security.
