# Savings Tracker

A simple, browser-based app to track savings (Mutual Funds, LIC, Bank Deposits, Sukanya Yojana) and daily expenses. Runs directly from GitHub Pages with no backend required.

## Features
- Add monthly savings and daily expenses
- Visualize savings and expenses with graphs
- Data stored locally (IndexedDB)
- Responsive, mobile-friendly UI
- Extensible for future mobile app development

## Getting Started
1. Clone or download this repo.
2. Open `index.html` in your browser, or deploy to GitHub Pages.

## Folder Structure
- `index.html`, `style.css`, `script.js`: Main app
- `db/`: Database schema
- `docs/`: Documentation
- `assets/`: Images/icons

## Extending
- Add new savings types by updating the form and chart logic in `script.js`.
- For mobile, consider using a framework like React Native or Flutter and reusing business logic.
